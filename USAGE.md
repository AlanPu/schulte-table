# 舒尔特表使用指南

## 🎮 如何开始游戏

### 第一步：启动应用
```bash
cd /Users/alan/workspaces/github.com/SchulteTable/SchulteTable
./gradlew :composeApp:run
```

### 第二步：选择难度和模式
1. 点击右上角的 **⚙️ 设置** 按钮
2. 选择难度：
   - **4×4** - 入门（16 个格子）
   - **5×5** - 基础（25 个格子）⭐ 推荐
   - **6×6** - 进阶（36 个格子）
3. 选择模式：
   - **数字** - 经典数字模式
   - **字母** - 英文字母 A-Z
   - **汉字** - 中文数字
   - **颜色** - 彩色方块
4. 点击 **确定**

### 第三步：开始游戏
1. 点击屏幕下方的 **"开始游戏"** 按钮
2. 计时器开始计时
3. 从 **1** 开始，依次点击格子中的数字
4. 点击正确后，"当前数字"会自动增加
5. 完成所有数字后，显示用时
6. 点击 **"重新开始"** 再次挑战

## 📋 游戏规则

### 基础规则
- ✅ 必须按顺序点击（1→2→3→...）
- ❌ 点击错误的数字不会有反应

- 🔄 可以随时重新开始

## 🎯 训练技巧

### 视觉训练
1. **扩大视野**：不要盯着单个数字，尝试一眼看到整个网格
2. **模式识别**：记住数字的大致位置
3. **快速扫描**：从上到下、从左到右系统性地搜索

### 提高速度
1. **放松眼睛**：保持适当的距离，不要离屏幕太近
2. **规律呼吸**：保持平稳的呼吸节奏
3. **多次练习**：每天练习 5-10 次，记录最佳成绩

### 专注力训练
1. **排除干扰**：在安静的环境中练习
2. **设定目标**：每次尝试打破自己的记录
3. **休息间隔**：每局之间休息 30 秒

## 🐛 常见问题

### Q: 点击数字没有反应？
A: 请确认：
- 游戏是否已经开始（点击"开始游戏"按钮）
- 是否按顺序点击（必须从 1 开始）
- 游戏是否已经结束（完成后需要重新开始）

### Q: 看不到"开始游戏"按钮？
A: 按钮显示在网格下方，如果窗口太小，尝试：
- 拉大窗口尺寸
- 滚动到底部查看

### Q: 如何改变难度？
A: 点击右上角的设置按钮，选择新的难度后点击确定

### Q: 计时器不走了？
A: 计时器只在游戏开始后运行，点击"开始游戏"启动
